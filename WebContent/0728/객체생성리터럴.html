1. circle객체 생성

- 속성 : 반지름 (rad)
- 메소드 : 면적 구하기 (getArea()), 둘레 구하기(getCircum())



1. Rect 객체 생성

- 속성 : 가로(width), 세로(height), 이름(name)
- 메소드 : 면적 구하기 (getArea()), 둘레 구하기(getCircum())



2. addr객체

- 정해진 속성과 메소드가 없다.
- 확인버튼 클릭하면 동적으로 속성과 메소드를 추가해서 사용할 수 있다



3. addr객체

- 정해진 속성과 메소드가 없다. var addr={ }
- 나중에 동적으로 속성과 메소드를 추가해서 사용할 수 있다
addr.name = "홍길동"
