작성팁

태그에 class속성과 id속성을 부여 할 수 있다.

class속성 :  여러개의 요소를 같은 형식의 스타일을 주고자 할 떄, css에서 .clss이름으로 지정한다.


id 속성 : d여러개의 요소 중 특정요소로 지정하고자 할 떄, 특벼한 수타일을 지정하고자 할 떄,
						  css에서 #이름으로 사용


box-sizing  : 브라우저의 크기를 조절할 때 내용도 같이 조절되러야 한다 

width : calc(30% - 1%) : 가로길이의 기준을 30%로 하고 줄일때 마다 1%씩 재계산되어 조절된다.
